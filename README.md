Microservice for main rental app: https://github.com/Jubiko31/rental-system-nestjs-docker.
Used streaming for file upload/download. For database used Redis.

## Installation

```bash
$ npm install
```

## Run 
```bash
$ npm run start
$ npm run start:dev
```
 And on main up run:
 ```bash
$ npm run listen
```
